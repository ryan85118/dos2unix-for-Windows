# dos2unix

## Installation

Install using [github](https://github.com/ryan85118/dos2unix.git):
```sh
git clone https://github.com/ryan85118/dos2unix.git
```

## Installation

Go to your dos2unix folder:
```sh
cd dos2unix
```

For execute script:
```sh
./dos2unix.ps1 -path <path>
```
> Note: `-path` is required for full path.
